# manual-proposal
작성했던 proposal들

```
인권과 인간의 존엄성

디지털정보공학과 201103277 조길상

1. 연구주제 : 인간은 왜 존엄한가

2. 연구목차 : 

   (1). 인간의 존엄성 정의
   (2). 인간의 존엄성 유래
   (3). 인간의 존엄성 내용
   (4). 인간의 존엄성 권리
   (5). 인간 존엄성에 대한 찬성 입장
   (6). 인간 존엄성에 대한 반대 입장
   (7). 인간은 존엄하다

3. 연구가치 : 헌법 제 10조 “모든 국민은 인간으로서의 존엄과 가치를 가지며 행복을 추구할권리를 가진다."
             에 명시될 만큼 인간의 존엄성에 대한 주제는 국민의 기본권과 모두에게 영향성을 미칠 중대한 주제이다.

4. 연구내용 및 방법 : 문헌 분석과 고찰 및 사례분석

참고문헌
인권철학입문, 크리스토프 멩케, 아른트 폴만 지음/정미라 주정립 옮김, 21세기북스 2012.
```
